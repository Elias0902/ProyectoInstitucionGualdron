¿Que es el Zorin?

Zorin OS es una distribución del sistema operativo GNU/Linux originaria de Irlanda y basada en Ubuntu. Está diseñada principalmente para usuarios novatos en GNU/Linux que también tienen experiencia con sistemas operativos Windows. Utiliza GNOME 3 y XFCE 4 como entornos de escritorio por defecto, pero personaliza su interfaz para facilitar la transición desde Windows y macOS.

El objetivo principal de Zorin OS es ofrecer una interfaz de usuario moderna, intuitiva y fácil de usar desde el principio, similar a la de sistemas como Windows 7, Windows XP y macOS. Es una de las pocas distribuciones de GNU/Linux, junto con Linux Mint, ChaletOS y Q4OS, cuya interfaz de usuario se asemeja más a la de Windows.

Además de su enfoque en la interfaz, Zorin OS es compatible con Wine y PlayOnLinux, lo que permite a los usuarios ejecutar aplicaciones de Windows, como Microsoft Office. También proporciona herramientas propias que simplifican diversas configuraciones relacionadas con la interfaz gráfica y la elección de software.

Introducción

Bienvenido al manual de usuario de Zorin OS. Descubre cómo utilizar este sistema operativo diseñado para una transición fácil desde Windows y macOS.

Instalación

Siga estos pasos para instalar Zorin OS:

  1. Descargue la imagen de Zorin OS desde el sitio oficial.
  2. Grabe la imagen en un USB o DVD.
  3. Arranque desde el medio de instalación y siga las instrucciones en pantalla.
  4. Complete el proceso de instalación.

Entorno de Escritorio

Zorin OS utiliza GNOME 3 y XFCE 4 como entornos de escritorio. Aprenda a navegar y personalizar su escritorio.

Compatibilidad

Zorin OS es compatible con aplicaciones de Windows. Descubra cómo utilizar Wine y PlayOnLinux para ejecutar software de Windows en Zorin OS.

Caracteristicas

Zorin OS es completamente gráfico, por lo que cuenta con un instalador gráfico. En cuanto a estabilidad y seguridad, se utiliza como base las versiones LTS del sistema principal Ubuntu. Utiliza sus propios repositorios de software, así como los repositorios de Ubuntu. Se puede acceder a estos repositorios a través de los comandos apt-get o apt install en el terminal de Linux, o un gestor de software con GUI que ofrece una experiencia similar a la de una tienda de aplicaciones para los usuarios que no desean utilizar el terminal.5​

El sistema operativo también cuenta con una variedad de diseños y temas de escritorio para personalizar el entorno de escritorio. Los temas permiten a los usuarios cambiar la interfaz para asemejarla a la de Microsoft Windows, macOS o Ubuntu,6​y permite que la interfaz sea familiar sin importar el sistema anterior del que provenga el usuario.7​8​9​ Al igual que con todos los entornos de escritorio basados en GNOME, el aspecto del escritorio puede modificarse utilizando extensiones para GNOME. 10​

Zorin OS incluye una selección bastante estándar de software en la versión normal, aunque existe mayor capacidad de elección con la versión premium (Zorin OS Pro). Mozilla Firefox es el navegador por defecto, sin embargo, Chromium, Opera y Midori pueden instalarse fácilmente utilizando Web Browser Manager o mediante Software. Otros programas que vienen por defecto instalados son: Rhythmbox como reproductor de música, GIMP como editor de imágenes y LibreOffice como suite ofimática.
